# Investment Calculator

Investment Calculator is a simple application built with Angular **18.2.14**.
The project was created **to learn and practice core Angular concepts**.

## Features

* Receive user input:

  * Initial Investment
  * Annual Investment
  * Expected Return
  * Duration
* Process investment data using a **dummy (non-real) calculation logic**
* Display results in a table with:

  * Year
  * Invested Value
  * Interest (Year)
  * Total Interest
  * Invested Capital

## Technologies

* Angular 18.2.14
* TypeScript
* HTML & CSS

## Purpose

This application was developed as a learning project to better understand Angular fundamentals such as components, data binding, directives, services, signals and data flow.

## How It Works

Users provide investment details through a form.
The application processes this data using a simplified (non-real) calculation logic and generates a yearly breakdown of the investment.
The results are displayed in a table format, making it easy to visualize growth over time.

## Getting Started

1. Clone the repository
2. Install dependencies

   ```bash
   npm install
   ```
3. Run the application

   ```bash
   ng serve
   ```
4. Open your browser at

   ```
   http://localhost:4200
   ```

## Notes

This is a basic project focused on learning and does not use real financial formulas or backend integration.
